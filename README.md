# 🔮 Magic Ball of Indecision — Магический Шар Неуверенности

Chrome-расширение для тревожных и нерешительных: карманный оракул, который берёт
на себя ответственность за твои решения. Задай вопрос — получи один из 50
уклончиво-честных ответов с щедрой порцией сарказма и неонового свечения.

## Возможности

- 🎱 3D-шар предсказаний с неоновым окном ответа в тёмной cyberpunk/mystic-теме
- 🌀 Анимация тряски шара при каждом вопросе
- 💨 Введённый вопрос «испаряется» — плавно растворяется в воздухе с размытием
- ✨ Ответ проявляется в окне шара с неоновой вспышкой
- 🗯️ 50 уникальных ответов на русском: саркастичные «да», циничные «нет» и дерзкие увиливания
- 😤 Отдельная реакция на пустой вопрос: «А спросить-то что хотел?»
- ⌨️ Отправка по кнопке или по Enter (Shift+Enter — перенос строки)

## Установка (Load unpacked)

1. Склонируй или скачай этот репозиторий.
2. Открой в Chrome страницу `chrome://extensions`.
3. Включи **Режим разработчика** (Developer mode) в правом верхнем углу.
4. Нажми **Загрузить распакованное расширение** (Load unpacked) и выбери папку репозитория.
5. Закрепи иконку шара на панели и спрашивай на здоровье.

## Стек

Manifest V3 + чистые HTML/CSS/JS — без сборщиков и зависимостей.
Всё, что лежит в репозитории, и есть готовое расширение.

## Структура

```
manifest.json   — манифест расширения (MV3)
popup.html      — разметка всплывающего окна
popup.css       — тема, 3D-шар и все анимации
popup.js        — логика: тряска, испарение, показ ответа
answers.js      — база из 50 ответов
icons/          — иконки 16/32/48/128
```
